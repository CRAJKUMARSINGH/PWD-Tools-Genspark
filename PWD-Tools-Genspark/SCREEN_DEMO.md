### 🚀 **Launch Commands**

#### **Windows Users:**
```cmd
# Install dependencies
INSTALL_DEPS.bat

# Launch application
START_APP.bat
```

#### **Unix/macOS Users:**
```bash
# Install dependencies
chmod +x install.sh
./install.sh

# Launch application
chmod +x start.sh
./start.sh
```

#### **Manual Launch:**
```bash
python main.py
```
